## HouseCare

Pagina web feita como trabalho final da disciplina de Desenvolvimento de software em nuvem

## Funcionalidades:

- CRUD de usuários
- CRUD de médicos
- CRUD de exames
- CRUD de especialidades
- CRUD de clínicas
- CRUD de agendamentos
- Autenticação de usuários via JWT (Login)

## Tecnologias:
- React (Vite)
- Typescript
- Node.js
- Tailwind.css
- Shadcn/ui
